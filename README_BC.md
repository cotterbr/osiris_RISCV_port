# Porting Osiris tool to RISC-V

----

This repo is a fork of the orignal Osiris tool build for x86.

The goal of this project is to use the Osiris fizzing framework but apply to RISC-V SonicBOOM architecture

### Building

Built with:

```bash
./build.sh INTEL
```


### Running

Run with:

```bash
./run.sh 0
```
